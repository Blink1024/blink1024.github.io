Windows系统下用Hugo+GitHub搭建个人博客

2020-01-21
学习之路
约 541 字 预计阅读 2 分钟

文章目录

安装Hugo

前往下载地址
下载 hugo_X.XX_Windows-64bit.zip
将其解压,得到 hogo.exe
将 hugo.exe 所在目录添加至系统环境变量

执行命令

hugo version

出现类似提示即安装成功

Hugo Static Site Generator v0.62.2-83E50184 windows/amd64 BuildDate: 2020-01-05T18:53:21Z

创建一个新站点

使用如下命令新建一个名为“quickstart”的网站：

hugo new site quickstart

添加一个主题

cd quickstart
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

编辑 config.toml 配置文件使用该主题
echo 'theme = "ananke"' >> config.toml

写一篇博客

执行命令

hugo new post/my-first-post.md

该命令会在quickstart/content/post 目录下新建一个my-first-post.md文件
编辑新建的markdown文件即可,注意将draft参数改为false

开启Hugo服务器

执行命令

hugo server -D

浏览器打开http://localhost:1313即可预览

部署到GitHub Pages

首先在GitHub上创建一个Repository，命名为：yourname.github.io
(yourname替换为自己的github用户名）

然后修改config.toml
baseURL = "https://yourname.github.io/"

执行命令

hugo

可看到根目录下出现/public文件夹,该文件夹的内容即Hugo生成的整个静态网站.

将/public的内容push到远程仓库.(注意替换yourname为自己的GitHub用户名)

cd public
git init
git add .
git commit -m "commit message"
git remote add origin https://github.com/yourname/yourname.github.io.git
git push -u origin master

完成后等待几分钟即可通过https://yourname.github.io/ 访问博客
